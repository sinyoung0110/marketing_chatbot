## 📦 제출 전 최종 확인사항

### 실행 테스트
```bash
# 1. 새 터미널에서 백엔드 실행
cd backend
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
cp .env.example .env
# .env 파일에 API 키 입력
python3 main.py

# 2. 새 터미널에서 프론트엔드 실행
cd frontend
npm install
npm start

# 3. 브라우저에서 http://localhost:3000 접속
# 4. 통합 워크플로우 테스트
```
