## 📊 페이지 1: 상세페이지 생성

### 기능:
- ✅ 상품 정보 입력 폼
- ✅ 플랫폼 선택 (쿠팡/네이버)
- ✅ 이미지 생성 옵션
- ✅ DALL-E로 실제 이미지 생성
- ✅ 마크다운/HTML 다운로드
- ✅ **SWOT + 3C 분석 보고서** 링크

### 사용 방법:
1. 상품명, 카테고리 입력
2. 키워드 입력 (쉼표로 구분)
3. 타겟 고객 입력
4. 플랫폼 선택
5. "생성하기" 클릭
6. 결과 확인 및 다운로드

### 결과물:
- `detail.md` - 마크다운 파일
- `detail.html` - HTML 파일
- `analysis.html` - **SWOT + 3C 분석 보고서**
- 이미지들 (main, usage)

## 📊 페이지 2: SWOT + 3C 분석

### 화면 구성:

```
┌─────────────────────────────────────┐
│  1️⃣ 경쟁사 검색                    │
│  [검색어 입력]                      │
│  [쿠팡] [네이버] [11번가]           │
│  [검색하기]                         │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│  2️⃣ 검색 결과 (15개)               │
│                [제외하고 재검색 (3)]│
│  ────────────────────────────────  │
│  ☑ 제품A - 쿠팡                     │
│      설명... [링크 열기 ↗]          │
│  ☐ 제품B - 네이버                   │
│      설명... [링크 열기 ↗]          │
│  ☑ 제품C - 쿠팡                     │
│      설명... [링크 열기 ↗]          │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│  3️⃣ 분석할 상품 정보               │
│  [상품명] [카테고리]                │
│  [키워드] [타겟 고객]               │
│  [SWOT + 3C 분석 실행]              │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│  ✅ 분석 완료!                      │
│  12개의 경쟁사 상품을 분석했습니다  │
│  [분석 보고서 열기]                 │
│  💡 핵심 인사이트:                  │
│  • 100% 국산 신선한 감자 사용       │
│  • 건강 간식 시장의 성장세          │
│  • 가격대: 3,000원 ~ 8,000원        │
└─────────────────────────────────────┘
```

### 주요 기능:

#### 1. 경쟁사 검색
- 검색어 입력
- 플랫폼 선택 (쿠팡/네이버/11번가)
- Tavily로 실시간 검색
- 최대 15개 결과

#### 2. 검색 결과 편집
- ✅ 각 URL 클릭하여 내용 확인
- ✅ 체크박스로 제외할 URL 선택
- ✅ "제외하고 재검색" 버튼
- ✅ 제외 개수 실시간 표시

#### 3. 상품 정보 입력
- 상품명
- 카테고리
- 키워드
- 타겟 고객

#### 4. 분석 실행
- 선택된 URL만 분석
- SWOT + 3C 자동 분석
- 가격 비교 및 최저가 탐색
- HTML 보고서 생성

## 💬 페이지 3: 마케팅 챗봇

### 화면 구성:

```
┌─────────────────────────┬─────────────┐
│ 💬 AI 마케팅 전략가      │  💡 추천 질문 │
│ ─────────────────────── │  ─────────  │
│ 🤖 AI: 안녕하세요!      │  [마케팅 전략]│
│    어떤 도움이...        │  [타겟 고객] │
│                         │  [가격 설정] │
│ 👤 YOU: 감자칩 마케팅   │  [SEO 키워드]│
│    전략 알려줘           │             │
│                         │  🚀 빠른 작업│
│ 🤖 AI: 감자칩의 경우... │  [키워드 추천]│
│    [키워드 추천]         │  [가격 전략] │
│    [SWOT 분석 →]        │  [타겟 분석] │
│                         │  ───────── │
│ [메시지 입력...] [전송] │  [상세페이지] │
│                         │  [SWOT 분석] │
│                         │             │
│                         │  📦 상품 정보│
│                         │  [상품명]   │
│                         │  [카테고리] │
└─────────────────────────┴─────────────┘
```

### 주요 기능:

#### 1. AI 대화
- 마케팅 전략 상담
- 타겟 고객 분석
- 가격 전략 제안
- SEO 키워드 추천
- 대화 히스토리 유지

#### 2. 빠른 작업 버튼
AI 응답에 자동으로 추가되는 버튼:
- **[키워드 추천]** → SEO 키워드 10개 제안
- **[가격 전략]** → 가격 포지셔닝 조언
- **[타겟 분석]** → 3가지 페르소나 분석
- **[상세페이지 생성 →]** → 페이지 1로 이동
- **[SWOT 분석 →]** → 페이지 2로 이동

#### 3. 추천 질문
클릭 한 번으로 입력:
- "마케팅 전략을 알려줘"
- "타겟 고객은 누구일까?"
- "가격은 어떻게 설정해야 할까?"
- "SEO 키워드를 추천해줘"

#### 4. 상품 정보 설정
우측 사이드바에서 상품 정보 입력:
- 상품명
- 카테고리

→ 빠른 작업 실행 시 사용됨

## 📝 사용 시나리오

### 시나리오 1: 신제품 런칭

```
1. [챗봇] "에어프라이어 감자칩 마케팅 전략 알려줘"
   → AI가 전략 제안
   → [키워드 추천] 클릭
   → SEO 키워드 10개 받음

2. [SWOT 분석] 탭으로 이동
   → "에어프라이어 감자칩" 검색
   → 쿠팡, 네이버 선택
   → 15개 결과 확인
   → 관련 없는 URL 3개 제외
   → 재검색
   → 분석 실행
   → HTML 보고서 다운로드

3. [상세페이지 생성] 탭으로 이동
   → 분석 결과 참고하여 입력
   → 상품명: "바삭 감자칩"
   → 카테고리: "간식"
   → 키워드: 받은 SEO 키워드 입력
   → 생성 클릭
   → MD/HTML 다운로드
```

### 시나리오 2: 경쟁사 조사만

```
1. [SWOT 분석] 직행
2. 검색 → 결과 확인 → 재검색
3. 분석 실행
4. 보고서 다운로드
```

### 시나리오 3: 빠른 상담

```
1. [챗봇] 직행
2. "이 상품 타겟 고객 누구야?" 질문
3. [타겟 분석] 버튼 클릭
4. 3가지 페르소나 받음
```
